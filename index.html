The Prosopography Hub is a site designed to host and deliver small prosopographical projects via web delivery and transformation to SNAP-compatible RDF. The prosopographical resources currently on this site originate from a range of existing, newly digitized, and experimental projects, and are all delivered via TEI XML (using the "Personography" module) and a generic XSLT transformation. Further functionality will be added in time.
