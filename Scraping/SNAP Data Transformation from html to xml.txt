# SNAP Data Transformation from HTML to XML

What it is:A python programme that replaces the html elements of the prosopographic data extracted with the previous script with xml ones.




Expected input:The openning of a file where the prosopographic data of the previous operation has been saved by the programmer. The usage of the .replace method to demonstrate the replacement of each html tag found in the file's content. 


What output it generates:The names, greek names and urls of each person or mythical creature demonstrated by XML elements instead of HTML ones


How to run it: Open file that contains extracted prosopographic data. Programme reads the content of the file. Demonstarte the replacements you want to make through 
the .replace method. Display the new form of data that you have created through the employment of a print statement.
